# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning](http://semver.org/spec/v2.0.0.html).

## [2.5.1] - 2026-01-21

### Fixed

- Adjusted Downloader launch delay to be applied when constructing the launcher.
- Avoided setting the Downloader visible flag when the window is not shown.

## [2.5.0] - 2026-01-17

### Added

- Added the Downloader tool for fetching large external assets.

## [2.4.4] - 2025-11-25

### Added

- Added signing for Unity 6.3.

## [2.4.3] - 2025-10-27

### Fixed

- Added a compatibility stub so VFX Graph Text Editor works in projects without
  VFX Graph installed.

## [2.4.2] - 2025-10-01

### Changed

- Enabled word wrapping in VFX Graph Text Editor.
